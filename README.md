# 소피
소피는 Software People의 줄임말로, 소프트웨어학과 학생들을 위한 애플리케이션입니다. 

자바를 기반으로 만들었으며, 안드로이드 스튜디오를 사용하였습니다.

## 주요 기능
+ 학과 공지사항 확인
+ 자유게시판, 프로젝트 팀매칭 게시판, 룸메이트 매칭 게시판
+ 스터디룸 예약
+ 학과 사무실 전화
